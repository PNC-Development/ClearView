About ClearView

History
ClearView was developed by Infrastructure Implementation Services (a division of Enterprise Production Support) over the last two years as a strategic initiative to eliminate touchpoints and decrease server build time. This tool is developed completely in-house and relies on no external applications or services to operate.
 
Modules
ClearView was designed to be modular meaning that is has individual peices that plug-in based on a user's permissions. Some of the current modules include...
  • Project Request and Approval
  • Service Request and Service Editor
  • Design Builder and Forecasting
  • Enterprise Reporting
  • On-Demand (workstations and servers)
  • Inventory Manager
